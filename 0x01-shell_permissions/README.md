This REDME gives descriptions about each script in this directory.

0-iam_betty = It switches current user to Betty.
1-who-am-i = It prints the username of the current user.
2-groups = It prints the groups the current user is part of.
3-new_owner = It changes the owner of file 'hello' to Betty.
4-empty = It creates an empty file 'hello'.
5-execute = It adds execute to existing user permission for file 'hello'.
6-multiple_permissions = It gives r-xr-xr-- to file 'hello'.
7-everybody = It grants execution permission to everyone.
8-James_Bond = It gives right to Other users only.
9-John_Doe = It adds certain permissions to file 'hello'.
10-mirror_permissions = It mirros the permissions of elloh to hello.
11-directories_permissions = It add execution permission to all subdirectories.
12-directory_permissions = It creates a new dir my_dir and gives 751 permission.
13-change_group = It changes the group of file 'hello' to school.
100-change_owner_and_group = It changes owner:group of everything in the working directory to vincent:staff.
101-symbolic_link_permissions = It changes the owner:group of _hello.
102-if_only = It changes the owner of file 'hello' to betty, if it is owned by guillaume
103-Star_Wars = It plays Star Wars on the terminal.
