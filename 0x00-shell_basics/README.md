This directory contains the following scripts and what they do;

0-current_working_directory = Prints Working Directory
100-lets_move = Moves all files starting with uppercase
101-clean_emacs = Deletes all ~(emacs) files
102-tree = Creates three directories of a single argument
103-commas = List the all the files of the current directories, seperating them with commas
10-back = changes to the previous directory
11-lists= List files from three different directories
12-file_type = Identify the type of files
13-symbolic_link = Creates a symbolic link
14-copy_html = Copies all the new html files to parent directory
1-listit = List the content of the current directory
2-bring_me_home = Changes the working directory to user's home directory
3-listfiles = List the content of the current directory in long format
4-listmorefiles = Displays the hidden files in long format
5-listfilesdigitonly = Displays the content in long formats, numerically
6-firstdirectory = It creates a directory name "my_first_directory"
7-movethatfile = Moves Betty to the above directory
8-firstdelete = Deletes betty
9-firstdirdeletion = Deletes the previously created directory
school.mgc = It detects school data when used with the file command
